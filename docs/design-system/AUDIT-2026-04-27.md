## Fluent surface nits

- **Tabular numbers**: Not verifiable from component source — Fluent's `fontFamilyNumeric` is available in `shared-overrides.ts` via `--cmtrace-font-family-mono`. Usage would need runtime inspection.
- **Severity row colors**: EntryGrid uses hardcoded rgba tints (see above) instead of theme-driven severity tokens. The Fluent surface has `--cmt-sev-error-bg` / `--cmt-sev-warning-bg` tokens that aren't being consumed.
- **Sentence-case labels**: All shell buttons use lowercase ("retry", "★ SAVE"). No violations found.
- **No emoji in product UI**: No emoji found in UI strings.

## Shell surface nits

- **All shell components read from `theme.*`**: Confirmed, except the `#9a7ef8` keyword color in KqlBar.
- **Pill states**: Exactly the six defined states are used (ok, okFallbacks, partial, failed, pending, stale). No rogue states.
- **Dotted texture**: `theme.pattern.dots` appears only in `Banner.tsx`. Correct.
- **KQL lexer colors**: Identifiers neutral (`theme.text`), operators/keywords accent (`theme.accent`), strings warm (`theme.pill.partial.fg`). The keyword purple (`#9a7ef8`) is the one exception — it's inline, not in theme.ts.
